# customer-segmentation-analysis
Customer segmentation analysis using SQL and Power BI to identify high-value customers, purchasing patterns, and city-wise revenue insights.

Customer Segmentation Analysis Dashboard

This project focuses on analyzing customer purchasing behavior using SQL and Power BI to identify high-value customers, revenue contribution by region, and purchasing patterns.

The dataset was stored and analyzed using PostgreSQL, and the insights were visualized through an interactive Power BI dashboard.

# Tools & Technologies
- SQL (PostgreSQL)
- pgAdmin
- Power BI
- Data Visualization
- Data Analysis

# Project Objectives
- Analyze customer purchase behavior
- Identify high-value customers
- Evaluate revenue contribution by city
- Understand order frequency and purchasing patterns

# SQL Analysis Performed
The following SQL queries were used for analysis:

1. Identifying high-value customers based on purchase amount
2. Calculating city-wise total revenue
3. Ranking customers by order frequency
4. Extracting data for visualization and reporting

# Dashboard Insights
The Power BI dashboard provides:

- City-wise revenue analysis
- Top customers based on purchase amount
- Purchase distribution across cities
- Key performance indicators such as total revenue and total customers

# Key Features
- Interactive visuals for data exploration
- KPI cards for quick business insights
- Bar charts for revenue comparison by city
- Tables for identifying top customers

## Business Value
This analysis helps businesses:

- Identify their most valuable customers
- Understand regional sales performance
- Improve customer targeting strategies
- Support data-driven business decisions
